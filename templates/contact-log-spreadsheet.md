# Contact Log Spreadsheet — Template Structure

The single highest-leverage piece of evidence in most parental-alienation cases. Years of dated cancellations on a single page change cases. Build the column structure today and update it weekly.

## Recommended columns

| Column | Description | Example |
|---|---|---|
| `Date` | The date of the scheduled contact / handover / event | 2026-05-14 |
| `Day` | Day of week (auto-computed) | Wed |
| `Scheduled time` | What the order / agreement specified | 18:00 pickup |
| `Outcome` | One of: `Occurred` / `Cancelled` / `No-show` / `Short` / `Other` | Cancelled |
| `Notified by` | Who cancelled + how | Mother / SMS @ 17:47 |
| `Reason given (verbatim)` | Direct quote from the other parent's message | "kid says she's tired" |
| `Source` | Filename / screenshot reference for the verbatim quote | `2026-05/sms-2026-05-14.png` |
| `My response (verbatim)` | What you replied — short, factual, non-escalatory | "Understood. Available tomorrow?" |
| `Witnesses present` | Anyone else who can confirm | Grandfather, school staff |
| `Notes` | Any context that doesn't fit above | First missed Wednesday in 6 weeks |

## Recommended file structure

```
case-file/
  evidence/
    contact-log.xlsx              ← the master spreadsheet
    2026-05/                       ← per-month folder of source files
      sms-2026-05-14.png
      email-2026-05-15.eml
      voicemail-2026-05-16.m4a
    2026-06/
      ...
```

## What this beats

A grieving parent's spoken testimony: "she keeps cancelling our visits." Possibly true; persuasively unprovable.

This spreadsheet with 47 dated entries quoting her own texts: a pattern the court can see.

Judges decide on patterns, not on speeches.

## Maintenance discipline

- Update within 24 hours of any incident — contemporaneous entries have far more weight than reconstructed ones.
- Never edit a past entry. Add a new row with a correction note.
- Back up monthly to cloud + external drive.
- Share read access with your lawyer (Google Sheets / OneDrive shared link).
