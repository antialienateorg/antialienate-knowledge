# Case Law

Cross-jurisdiction reference for parental-alienation and custody case law.

## Index

- **[Cross-jurisdiction 2024–2026 rulings consolidation](./2024-2026-rulings.md)** — ~60 jurisdictions covered: ECHR, EU, Commonwealth, Latin America, Asia, US. Organised by court tier and region.

See also:

- [/landmark-cases/](../landmark-cases/) — the foundational cases everyone should be able to cite from memory (Troxel v Granville, Strand Lobben, Re C [2023], etc.)
- [/jurisdictions/](../jurisdictions/) — per-jurisdiction legal-reference pages with statutes and procedural mechanisms.
- [/tools/legal-research.md](../tools/legal-research.md) — free authoritative databases for primary-source verification.

## Verification protocol

Every citation in `/case-law/` and `/landmark-cases/` should ideally link to a primary source (court database, official reporter). Where citations remain unlinked, treat them as research leads rather than verified holdings — verify against primary source before using in any filing.
