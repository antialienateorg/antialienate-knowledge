# Community — Reddit + Advocacy Orgs + Peer Support

The peer-to-peer and organisational layer of the parental-alienation ecosystem. These are the places to find other parents working through the same issue, to plug into advocacy efforts, and to participate in legislative reform.

## Index

- [Reddit subreddits](./reddit-subreddits.md) — direct topic + adjacent communities, with how-to-use guidance.
- [Advocacy organisations](./advocacy-orgs.md) — international + regional advocacy and support organisations.

## Note on framing

Across the community ecosystem, you'll find competing framings — clinical / fathers'-rights / safeguarding-first / equal-parenting. AntiAlienate's framing is empathy-first, evidence-anchored, jurisdiction-agnostic, both-gender-targeted. We catalogue all credible perspectives without endorsement of any single orientation.
