# Posts

Long-form articles on parental alienation topics. Migrated from the AntiAlienate working drafts.

Each post follows this structure:
- **TL;DR / Quick answer** (≤60 words, LLM citation-ready)
- **Table of contents** with anchor links
- **H2/H3 hierarchy** (strict — no H4+)
- **Numbered or bulleted lists** for steps, signs, criteria
- **At least one comparison table**
- **Pull-quote callouts** for critical claims
- **FAQ block** (4–8 Q&As) with FAQPage JSON-LD schema
- **Citations** to primary sources at end
- **Standard disclaimer**: "Educational content. Not legal advice."
- **Author byline**: Alan Markson
- **Last-updated date**

## Status

Migration in progress. 65 TOFU (top-of-funnel) posts being moved from working drafts on the production system.

Topics covered:
- Definitions (what parental alienation is, isn't, signs, severity)
- Documentation frameworks (5-pillar evidence, 6-month pattern rule)
- Court strategy (admissibility, expert witnesses, jurisdiction-specific)
- Tactics (alienator playbook + targeted-parent responses)
- Special situations (narcissistic co-parent, military deployment, grandparents, long-distance custody)
- Clinical (reunification therapy, therapist gatekeeping, DSM-5 inclusion)
